What is JavaScript?

JavaScript is a versatile, high-level programming language that enables interactive web pages. Originally created for browsers, JavaScript now runs on servers (Node.js), mobile apps, and desktop applications. It's essential for modern web development, handling everything from simple animations to complex single-page applications.

// JavaScript Interactive Examples

// 1. DOM Manipulation and Event Handling
document.addEventListener('DOMContentLoaded', function() {
    const button = document.getElementById('interactive-btn');
    const counter = document.getElementById('counter');
    const colorBox = document.getElementById('color-box');
    
    let count = 0;
    const colors = ['#ff6b6b', '#4ecdc4', '#45b7d1', '#96ceb4', '#ffeaa7'];
    
    button.addEventListener('click', function() {
        // Update counter
        count++;
        counter.textContent = count;
        
        // Change background color
        const randomColor = colors[Math.floor(Math.random() * colors.length)];
        colorBox.style.backgroundColor = randomColor;
        
        // Add animation
        colorBox.style.transform = 'scale(1.1)';
        setTimeout(() => {
            colorBox.style.transform = 'scale(1)';
        }, 200);
    });
});

// 2. Async/Await and API Calls
class UserManager {
    constructor() {
        this.users = [];
    }
    
    async fetchUsers() {
        try {
            const response = await fetch('https://jsonplaceholder.typicode.com/users');
            if (!response.ok) {
                throw new Error('Failed to fetch users');
            }
            this.users = await response.json();
            this.displayUsers();
        } catch (error) {
            console.error('Error fetching users:', error);
            this.showError('Failed to load users. Please try again.');
        }
    }
    
    displayUsers() {
        const userList = document.getElementById('user-list');
        userList.innerHTML = '';
        
        this.users.forEach(user => {
            const userCard = document.createElement('div');
            userCard.className = 'user-card';
            userCard.innerHTML = `
                <h3>${user.name}</h3>
                <p>Email: ${user.email}</p>
                <p>Company: ${user.company.name}</p>
            `;
            userList.appendChild(userCard);
        });
    }
}

// 3. ES6+ Features and Modern JavaScript
const TodoApp = {
    todos: [],
    
    // Arrow functions and destructuring
    addTodo: (text) => {
        const newTodo = {
            id: Date.now(),
            text,
            completed: false,
            createdAt: new Date().toISOString()
        };
        TodoApp.todos.push(newTodo);
        TodoApp.render();
    },
    
    // Template literals and array methods
    render() {
        const todoContainer = document.getElementById('todo-container');
        const todoHTML = this.todos.map(({id, text, completed}) => `
            <div class="todo-item ${completed ? 'completed' : ''}">
                <input type="checkbox" ${completed ? 'checked' : ''} 
                       onchange="TodoApp.toggleTodo(${id})">
                <span>${text}</span>
                <button onclick="TodoApp.deleteTodo(${id})">Delete</button>
            </div>
        `).join('');
        
        todoContainer.innerHTML = todoHTML;
    }
};

Explanation of the Code:

  • 🖱️ DOM Manipulation & Event Handling: Listens for button clicks, updates a counter, changes colors, and adds a scaling animation.
  • 🌐 Async/Await & API Calls: Fetches user data from an API, handles errors gracefully, and displays results dynamically.
  • 🧩 ES6+ Features: Uses arrow functions, destructuring, template literals, and array methods for cleaner, modern syntax.
  • 📝 Todo App: Demonstrates adding, rendering, and toggling tasks with a responsive UI using JavaScript objects and methods.
  • ⚡ Reusable Patterns: Combines interactivity, asynchronous programming, and modern JavaScript techniques into practical mini-apps.
