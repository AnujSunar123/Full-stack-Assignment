HTML

HyperText Markup Language - the foundation of every website.

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure and content of a webpage using elements and tags. HTML elements tell the browser how to display content - whether it's a heading, paragraph, image, link, or any other type of content.

Key HTML Elements:

  • <h1> - <h6> Headings
  • <p> Paragraphs
  • <a> Links
  • <img> Images
  • <div> Containers

Semantic Elements:

  • <header> Page header
  • <nav> Navigation
  • <main> Main content
  • <section> Content sections
  • <footer> Page footer
HTML Example 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <header>
        <h1>Welcome to My Website</h1>
    </header>
</body>
</html>

Explanation of the Example:

  • • The <!DOCTYPE html> defines the document type.
  • • The <head> includes metadata.
  • • The <body> contains visible content.
  • • The <header> includes title/navigation.
  • • Semantic elements improve SEO & accessibility.
