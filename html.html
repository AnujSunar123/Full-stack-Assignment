HTML

HyperText Markup Language - the foundation of every website.

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure and content of a webpage using elements and tags. HTML elements tell the browser how to display content - whether it's a heading, paragraph, image, link, or any other type of content.

Key HTML Elements:

  • • <h1> - <h6> Headings
  • • <p> Paragraphs
  • • <a> Links
  • • <img> Images
  • • <div> Containers

Semantic Elements:

  • • <header> Page header
  • • <nav> Navigation
  • • <main> Main content
  • • <section> Content sections
  • • <footer> Page footer
HTML 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <header>
        <h1>Welcome to My Website</h1>
        <nav>
            <ul>
                <li><a href="#about">About</a></li>
                <li><a href="#services">Services</a></li>
                <li><a href="#contact">Contact</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <section id="about">
            <h2>About Me</h2>
            <p>I'm a web developer passionate about creating 
               amazing user experiences.</p>
            <img src="profile.jpg" alt="Profile picture" width="200">
        </section>
        
        <section id="services">
            <h2>My Services</h2>
            <div>
                <h3>Web Development</h3>
                <p>Custom websites built with modern technologies.</p>
            </div>
        </section>
    </main>
    
    <footer>
        <p>© 2024 My Website. All rights reserved.</p>
    </footer>
</body>
</html>

Explanation of the Example:

  • • The <!DOCTYPE html> defines the document type and version of HTML.
  • • The <head> section includes metadata like title and character encoding.
  • • The <body> contains the visible content of the webpage.
  • • The <header> includes a title and navigation menu.
  • • The <main> contains structured content using <section>.
  • • The <footer> displays copyright information.
  • • Semantic elements improve accessibility and SEO by giving meaning to the content.
