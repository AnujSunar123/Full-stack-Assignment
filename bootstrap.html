Bootstrap

The world's most popular CSS framework for responsive web development.

What is Bootstrap?

Bootstrap is a powerful, mobile-first CSS framework that provides pre-built components, utilities, and a responsive grid system. It allows developers to quickly create professional-looking websites without writing extensive custom CSS. Bootstrap includes JavaScript components for interactive elements like modals, dropdowns, and carousels.

Bootstrap Components:

  • • Navigation bars
  • • Cards and modals
  • • Forms and buttons
  • • Alerts and badges
  • • Carousel and accordions

Key Features:

  • • Responsive grid system
  • • Mobile-first approach
  • • Extensive utility classes
  • • Customizable themes
  • • JavaScript plugins
HTML 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Example</title>
    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" 
          rel="stylesheet">
</head>
<body>
    <!-- Navigation -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">My Portfolio</a>
            <button class="navbar-toggler" type="button" data-bs-toggle="collapse" 
                    data-bs-target="#navbarNav">
                <span class="navbar-toggler-icon"></span>
            </button>
            <div class="collapse navbar-collapse" id="navbarNav">
                <ul class="navbar-nav ms-auto">
                    <li class="nav-item">
                        <a class="nav-link" href="#about">About</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#projects">Projects</a>
                    </li>
                </ul>
            </div>
        </div>
    </nav>

    <!-- Hero Section -->
    <div class="container my-5">
        <div class="row">
            <div class="col-lg-8 mx-auto text-center">
                <h1 class="display-4 fw-bold text-primary">Welcome to My Portfolio</h1>
                <p class="lead mb-4">I create beautiful, responsive websites using modern technologies.</p>
                <button class="btn btn-primary btn-lg me-3">View Projects</button>
                <button class="btn btn-outline-primary btn-lg">Contact Me</button>
            </div>
        </div>
    </div>

    <!-- Cards Section -->
    <div class="container mb-5">
        <div class="row g-4">
            <div class="col-md-4">
                <div class="card h-100 shadow-sm">
                    <div class="card-body">
                        <h5 class="card-title">Web Development</h5>
                        <p class="card-text">Creating responsive and modern websites with the latest technologies.</p>
                        <a href="#" class="btn btn-primary">Learn More</a>
                    </div>
                </div>
            </div>
        </div>
    </div>

    <!-- Bootstrap JS -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>

Explanation of the Code:

  • ✅ Includes Bootstrap 5 CSS and JS via CDN for quick setup.
  • 🎨 Responsive navigation bar (`navbar`, `navbar-expand-lg`, `bg-primary`) that collapses into a toggle menu on smaller screens.
  • 📱 Hero section with Bootstrap utility classes (`container`, `row`, `col-lg-8`, `text-center`) for layout and alignment.
  • ✨ Uses typography classes (`display-4`, `fw-bold`, `lead`, etc.) for styled headings and text.
  • 🖱️ Buttons styled with Bootstrap classes (`btn`, `btn-primary`, `btn-outline-primary`, `btn-lg`) for consistency.
  • 🃏 Card component (`card`, `shadow-sm`, `h-100`) showing Bootstrap’s grid (`row g-4`, `col-md-4`) for responsive design.
  • ⚡ No custom CSS required — Bootstrap’s utility classes handle layout, spacing, and responsiveness.
