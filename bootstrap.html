Bootstrap

The world's most popular CSS framework for responsive web development.

What is Bootstrap?

Bootstrap is a powerful, mobile-first CSS framework that provides pre-built components, utilities, and a responsive grid system. It allows developers to quickly create professional-looking websites without writing extensive custom CSS. Bootstrap includes JavaScript components for interactive elements like modals, dropdowns, and carousels.

Bootstrap Components:

  • • Navigation bars
  • • Cards and modals
  • • Forms and buttons
  • • Alerts and badges
  • • Carousel and accordions

Key Features:

  • • Responsive grid system
  • • Mobile-first approach
  • • Extensive utility classes
  • • Customizable themes
  • • JavaScript plugins
HTML Example 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bootstrap Example</title>
    <!-- Bootstrap CSS -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" 
          rel="stylesheet">
</head>
<body>
    <!-- Navigation -->
    <nav class="navbar navbar-expand-lg navbar-dark bg-primary">
        <div class="container">
            <a class="navbar-brand" href="#">My Portfolio</a>
            <button class="navbar-toggler" type="button" data-bs-toggle="collapse" 
                    data-bs-target="#navbarNav">
                <span class="navbar-toggler-icon"></span>
            </button>
            <div class="collapse navbar-collapse" id="navbarNav">
                <ul class="navbar-nav ms-auto">
                    <li class="nav-item">
                        <a class="nav-link" href="#about">About</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#projects">Projects</a>
                    </li>
                </ul>
            </div>
        </div>
    </nav>
</body>
</html>

Explanation of the Code:

  • ✅ Includes Bootstrap 5 CSS and JS via CDN for quick setup.
  • 🎨 Responsive navbar that collapses on smaller screens.
  • 📱 Hero section uses Bootstrap grid (`container`, `row`, `col`) for layout.
  • ✨ Typography classes (`display-4`, `lead`, `fw-bold`) style headings and text.
  • 🖱️ Buttons use Bootstrap classes (`btn`, `btn-primary`, `btn-outline-primary`).
  • 🃏 Card component (`card`, `shadow-sm`) demonstrates Bootstrap’s grid for responsive design.
